# 键盘快捷键

TabRest 提供键盘快捷键，无需打开弹出窗口即可快速管理标签页。

## 默认快捷键

| 快捷键 | 操作 | 描述 |
|----------|--------|-------------|
| `Alt+Shift+D` | 卸载当前标签页 | 卸载当前活动的标签页 |
| `Alt+Shift+O` | 卸载其他标签页 | 卸载除当前之外的所有标签页 |
| `Alt+Shift+→` | 卸载右侧 | 卸载右侧的所有标签页 |
| `Alt+Shift+←` | 卸载左侧 | 卸载左侧的所有标签页 |

**注意**：在 macOS 上，快捷键可能使用 `Option` 而非 `Alt`。

## 如何自定义快捷键

Chrome 允许您自定义扩展的键盘快捷键：

### 分步指南

1. **打开快捷键页面**：在浏览器中导航到 `chrome://extensions/shortcuts`

2. **找到 TabRest**：滚动到 TabRest 部分

3. **编辑快捷键**：
   - 点击铅笔图标或快捷键字段
   - 按下您想要的组合键
   - 点击字段外部即可保存

4. **清除快捷键**：点击 X 图标以移除快捷键

### 快捷键要求

Chrome 对扩展快捷键强制执行以下规则：

- **必须包含修饰键**：至少包含 `Ctrl`、`Alt`、`Shift`、`Command`（macOS）之一
- **不能使用**：浏览器保留的快捷键（例如 `Ctrl+T`、`Ctrl+W`）
- **推荐**：使用 `Alt+Shift+[键]` 以避免与网页快捷键冲突

### 自定义快捷键示例

常用替代方案：

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+Shift+D` | 卸载当前标签页 |
| `Alt+U` | 卸载其他标签页 |
| `Ctrl+Alt+→` | 卸载右侧 |
| `Ctrl+Alt+←` | 卸载左侧 |

## 故障排除

### 快捷键不起作用

**问题**：按下快捷键没有任何反应。

**解决方案**：

1. **检查快捷键页面**：访问 `chrome://extensions/shortcuts` 并确认 TabRest 快捷键已设置

2. **检查冲突**：如果快捷键显示为灰色或显示"已被 Chrome 使用"，请选择其他组合

3. **重新加载扩展**：进入 `chrome://extensions`，找到 TabRest，然后点击刷新图标

4. **在不同环境中测试**：某些快捷键在 Chrome 内部页面（`chrome://`、`chrome-extension://`）上不起作用

### 快捷键冲突

如果您的自定义快捷键与以下项冲突：

- **其他扩展**：在 `chrome://extensions/shortcuts` 中禁用冲突扩展的快捷键
- **网页快捷键**：TabRest 快捷键优先于网页键盘事件
- **Chrome 浏览器快捷键**：您无法覆盖这些快捷键（例如 `Ctrl+T`、`Ctrl+N`）

### 平台差异

| 平台 | 修饰键 | 示例 |
|----------|--------------|---------|
| Windows/Linux | `Alt` | `Alt+Shift+D` |
| macOS | `Option` 或 `Alt` | `Option+Shift+D` |
| ChromeOS | `Alt` | `Alt+Shift+D` |

**注意**：某些 macOS 键盘将 `Alt` 标记为 `Option`。它们是同一个键。

## 最佳实践

### 推荐快捷键

- **卸载当前标签页**（`Alt+Shift+D`）：用完一个标签页时快速访问
- **卸载其他标签页**（`Alt+Shift+O`）：清理除焦点之外的所有内容
- **卸载右侧**（`Alt+Shift+→`）：清除从当前标签页打开的标签页

### 何时使用快捷键 vs 弹出窗口

**使用快捷键** 适用于：
- 快速的单一操作
- 工作流集成
- 重度依赖键盘的工作流

**使用弹出窗口** 适用于：
- 查看统计信息
- 管理白名单
- 卸载标签组
- 调整设置

## 无障碍

键盘快捷键使 TabRest 对以下人群更易于使用：

- **高级用户**：比点击图标更快
- **屏幕阅读器用户**：无需鼠标即可导航
- **行动不便者**：尽量减少指针移动

所有弹出窗口操作仍可通过标准键盘导航（`Tab`、`Enter`、`Space`）访问。
