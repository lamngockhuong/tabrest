# キーボードショートカット

TabRest は、ポップアップを開かずに素早くタブを管理するためのキーボードショートカットを提供します。

## デフォルトのショートカット

| ショートカット | アクション | 説明 |
|----------|--------|-------------|
| `Alt+Shift+D` | 現在のタブをアンロード | アクティブなタブをアンロードします |
| `Alt+Shift+O` | 他のタブをアンロード | 現在のタブ以外をすべてアンロードします |
| `Alt+Shift+→` | 右をアンロード | 右側のすべてのタブをアンロードします |
| `Alt+Shift+←` | 左をアンロード | 左側のすべてのタブをアンロードします |

**注意**: macOS では、ショートカットに `Alt` の代わりに `Option` を使用する場合があります。

## ショートカットのカスタマイズ方法

Chrome では拡張機能のキーボードショートカットをカスタマイズできます:

### ステップバイステップガイド

1. **ショートカットページを開く**: ブラウザで `chrome://extensions/shortcuts` に移動します

2. **TabRest を見つける**: TabRest のセクションまでスクロールします

3. **ショートカットを編集**:
   - 鉛筆アイコンまたはショートカットフィールドをクリックします
   - 希望のキーの組み合わせを押します
   - フィールドの外側をクリックして保存します

4. **ショートカットをクリア**: X アイコンをクリックしてショートカットを削除します

### ショートカットの要件

Chrome は拡張機能のショートカットに次のルールを課しています:

- **修飾キーが必須**: `Ctrl`、`Alt`、`Shift`、`Command`（macOS）のうち少なくとも 1 つ
- **使用できないもの**: ブラウザが予約しているショートカット（例: `Ctrl+T`、`Ctrl+W`）
- **推奨**: ウェブページのショートカットとの競合を避けるため `Alt+Shift+[キー]` を使用します

### カスタムショートカットの例

人気の代替案:

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+D` | 現在のタブをアンロード |
| `Alt+U` | 他のタブをアンロード |
| `Ctrl+Alt+→` | 右をアンロード |
| `Ctrl+Alt+←` | 左をアンロード |

## トラブルシューティング

### ショートカットが動作しない

**問題**: ショートカットを押しても何も起こりません。

**解決策**:

1. **ショートカットページを確認**: `chrome://extensions/shortcuts` を開き、TabRest のショートカットが設定されているか確認します

2. **競合を確認**: ショートカットがグレーアウトしているか「Chrome が使用中」と表示されている場合は、別の組み合わせを選びます

3. **拡張機能を再読み込み**: `chrome://extensions` を開き、TabRest を見つけて更新アイコンをクリックします

4. **別のコンテキストでテスト**: 一部のショートカットは Chrome の内部ページ（`chrome://`、`chrome-extension://`）では動作しません

### ショートカットの競合

カスタムショートカットが次のものと競合する場合:

- **他の拡張機能**: `chrome://extensions/shortcuts` で競合する拡張機能のショートカットを無効にします
- **ウェブページのショートカット**: TabRest のショートカットはウェブページのキーボードイベントより優先されます
- **Chrome ブラウザのショートカット**: これらは上書きできません（例: `Ctrl+T`、`Ctrl+N`）

### プラットフォームの違い

| プラットフォーム | 修飾キー | 例 |
|----------|--------------|---------|
| Windows/Linux | `Alt` | `Alt+Shift+D` |
| macOS | `Option` または `Alt` | `Option+Shift+D` |
| ChromeOS | `Alt` | `Alt+Shift+D` |

**注意**: 一部の macOS キーボードでは `Alt` が `Option` と表記されています。これらは同じキーです。

## ベストプラクティス

### 推奨ショートカット

- **現在のタブをアンロード**（`Alt+Shift+D`）: タブを使い終わったときの素早いアクセス
- **他のタブをアンロード**（`Alt+Shift+O`）: 集中対象以外をすべて整理
- **右をアンロード**（`Alt+Shift+→`）: 現在のタブから開いたタブをクリア

### ショートカットとポップアップの使い分け

**ショートカットを使う**のは:
- 素早い単一のアクション
- ワークフローへの統合
- キーボード中心のワークフロー

**ポップアップを使う**のは:
- 統計の表示
- ホワイトリストの管理
- タブグループのアンロード
- 設定の調整

## アクセシビリティ

キーボードショートカットは、次のような場合に TabRest をアクセシブルにします:

- **パワーユーザー**: アイコンをクリックするより速い
- **スクリーンリーダーの利用者**: マウスなしで操作
- **可動性に制約のある方**: ポインターの移動を最小限に

すべてのポップアップアクションは、標準的なキーボード操作（`Tab`、`Enter`、`Space`）で引き続きアクセスできます。
