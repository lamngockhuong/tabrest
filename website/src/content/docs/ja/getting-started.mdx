# TabRest をはじめる

TabRest は、非アクティブなタブを自動的にアンロードしてメモリを解放するブラウザ拡張機能です。Chrome、Microsoft Edge、Brave、Opera、Vivaldi、Arc など、**すべての Chromium ベースのブラウザ**で動作します。

## インストール

### ブラウザのウェブストアから

**Chrome をお使いの方:**
1. [Chrome ウェブストアの TabRest ページ](https://chromewebstore.google.com/detail/tabrest-rest-your-tabs-fr/agajndkecodedlklmpnjgikglkpeopib)を開きます
2. **Chrome に追加**をクリックします

**Edge をお使いの方:**
1. [Microsoft Edge アドオンの TabRest ページ](https://microsoftedge.microsoft.com/addons)を開きます（近日公開予定）
2. **入手**をクリックします

**Brave、Opera、Vivaldi、Arc をお使いの方:**
- Chrome ウェブストアから直接インストールできます（これらのブラウザは Chrome 拡張機能に対応しています）

3. 権限リクエストを承認します
4. ブラウザのツールバーに TabRest アイコンが表示されます

### ソースから（開発者向け）

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/lamngockhuong/tabrest.git
   cd tabrest
   ```

2. ブラウザを開き、拡張機能ページに移動します:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`
   - Opera: `opera://extensions`
   - Vivaldi: `vivaldi://extensions`

3. **デベロッパーモード**を有効にします（右上のトグル）

4. **パッケージ化されていない拡張機能を読み込む**をクリックし、`tabrest` フォルダを選択します

5. 拡張機能が有効になりました

## 最初のステップ

インストール後、TabRest はデフォルト設定ですぐに動作します:

- **自動アンロードタイマー**: 30 分間の非アクティブ状態
- **メモリしきい値**: RAM 使用率 80%
- **保護対象のタブ**: 固定タブ、音声を再生しているタブ、未保存フォームのあるタブ
- **ホワイトリスト**: YouTube、Gmail、Google ドキュメント（初期設定済み）

### クイック設定

1. **設定を開く**: TabRest アイコン → ⚙️ 設定ボタンをクリックします
2. **タイマーを調整**: 5 分から 4 時間の間で選択します
3. **メモリしきい値を設定**: RAM 使用率 60〜95% のトリガーを設定します
4. **ホワイトリストに追加**: 特定のサイトを自動アンロードから保護します

### 初めての手動アンロード

手動でタブをアンロードして、動作を確認してみましょう:

1. 複数のタブを開きます
2. ツールバーの TabRest アイコンをクリックします
3. アクションを選択します:
   - **現在のタブをアンロード**: アクティブなタブをアンロードします
   - **他のタブをアンロード**: 現在のタブ以外をすべてアンロードします
   - **左/右をアンロード**: 片側のタブをアンロードします

アンロードされたタブはタブバーに表示されたままですが、クリックすると再読み込みされます。

## タブがアンロードされると何が起こるか

TabRest はブラウザのネイティブな `tabs.discard()` API を使用します。アンロードされたタブは:

- ✅ タイトルとファビコン付きでタブバーに表示されたままになります
- ✅ スクロール位置とフォームデータを保持します
- ✅ クリックすると即座に再読み込みされます
- ✅ 非アクティブ中はメモリを解放します
- ❌ バックグラウンドスクリプトの実行やメディアの再生はできません

## サイドパネルモードを使う

ポップアップを開くのではなく、TabRest を常に表示しておきたい場合は:

1. TabRest アイコン → **⚙️ 設定**ボタンをクリックします
2. **ツールバーアクション**セクションまでスクロールします
3. **サイドパネルで開く**をオンに切り替えます
4. 次回のクリックで TabRest がブラウザのサイドバーに開きます
5. タブを切り替えてもパネルは開いたままになります
6. いつでもオフに切り替えてポップアップモードに戻せます

**注意**: サイドパネルには Chrome 114 以降が必要です。

## 自動で開く変更履歴

TabRest が新機能を伴って更新されると（マイナーまたはメジャーバージョン）、変更履歴が自動的に開き、新しい内容を確認できます。パッチ更新（バグ修正）では、不要な通知を避けるため変更履歴は開きません。
