# Features

TabRest provides multiple ways to manage your tabs and free up memory.

## Automatic Unloading

### Timer-Based Unloading

Automatically unload tabs after a period of inactivity:

| Timer Option | Use Case |
|--------------|----------|
| 15 minutes   | Aggressive memory saving |
| 30 minutes   | Default balanced mode |
| 1 hour       | Light users |
| 2 hours      | Keep tabs longer |
| 4 hours      | Minimal auto-unload |

**How it works**: TabRest tracks when you last interacted with each tab (clicks, scrolling, focus). After the configured time, inactive tabs are unloaded.

### Memory Threshold Unloading

Unload tabs when system RAM usage exceeds a threshold:

- **Range**: 60-90% memory usage
- **Default**: 80%
- **Check interval**: Every 30 seconds

When triggered, TabRest unloads the least recently used (LRU) tabs until memory drops below the threshold.

### Startup Unload

Free memory immediately when Chrome starts:

- **Default**: Enabled
- **Behavior**: Unloads all tabs except the active one on browser launch
- **Useful for**: Restoring many tabs from a previous session

### Tab Count Threshold

Only start auto-unloading when you have enough inactive tabs:

- **Range**: 3-20 tabs
- **Default**: 6 tabs
- **Disabled**: Set to 0 to always auto-unload

**Use case**: Prevents aggressive unloading when you only have a few tabs open.

### Idle State Only

Only auto-unload tabs when your computer is idle:

- **Default**: Disabled
- **Threshold**: 2-15 minutes of inactivity
- **Detection**: Uses Chrome's idle API (detects mouse/keyboard activity)

**Use case**: Never interrupt your active browsing session.

### Per-Tab Memory Threshold

Automatically unload individual tabs that consume excessive memory:

- **Range**: 100MB - 1GB (JavaScript heap)
- **Default**: Disabled
- **Note**: Only measures JS memory, not total tab memory

**Use case**: Target memory-heavy tabs specifically rather than by age.

### Skip When Offline

Prevent tabs from being discarded when network is unavailable:

- **Default**: Enabled
- **Behavior**: Pauses auto-unload when `navigator.onLine` is false
- **Why**: Discarded tabs can't reload without network, avoiding frustration

## Manual Controls

Control tab unloading on-demand from the popup:

| Action | Description |
|--------|-------------|
| **Unload Current** | Unload the active tab |
| **Unload Others** | Unload all tabs except current |
| **Unload Left** | Unload all tabs to the left of current |
| **Unload Right** | Unload all tabs to the right of current |

Access via the TabRest icon in the toolbar or [keyboard shortcuts](/docs/keyboard-shortcuts).

### Toolbar Click Action

Customize what happens when you click the extension icon:

| Option | Behavior |
|--------|----------|
| **Open popup** | Default - shows tab list and quick settings |
| **Discard current** | Immediately unload current tab |
| **Discard others** | Immediately unload all other tabs |

**Use case**: Power users who want one-click discard without opening popup.

## Power Mode

Adjust auto-unload aggressiveness based on your situation:

| Mode | Timer Multiplier | Memory Offset |
|------|------------------|---------------|
| **Battery Saver** | 0.5x (faster) | -10% (lower threshold) |
| **Normal** | 1.0x (default) | 0% |
| **Performance** | 2.0x (slower) | +10% (higher threshold) |

**Example**: If your timer is set to 30 minutes:
- Battery Saver: 15 minutes
- Normal: 30 minutes
- Performance: 60 minutes

## Tab Protection

Protect specific tabs from automatic unloading:

### Pinned Tabs
- **Default**: Protected from auto-unload
- **Configurable**: Enable "Unload pinned tabs" in settings to override

### Audio/Video Tabs
- **Default**: Protected when playing media
- **Detection**: Checks `tab.audible` property
- **Configurable**: Disable "Protect tabs playing audio/video" to allow unloading

### Form Tabs
- **Default**: Protected when forms have unsaved data
- **Detection**: Content script checks for modified input fields
- **Configurable**: Disable "Protect tabs with unsaved forms" to allow unloading

### Whitelist
- **Default**: YouTube, Gmail, Google Docs, Miro, Figma, Notion
- **Custom**: Add any site to prevent auto-unload
- **Details**: See [Whitelist Configuration](/docs/whitelist-config)

### Snooze Tabs/Domains

Temporarily protect specific tabs or entire domains from auto-discard:

| Snooze Option | Duration |
|---------------|----------|
| Tab snooze | 30 min, 1 hour, 2 hours |
| Domain snooze | 1 hour |

**Access**: Right-click on page → Snooze options, or use ⏸️ button in popup tab list.

**Use case**: Working intensively on a site without adding it to permanent whitelist.

## Tab Groups

Unload entire tab groups at once:

1. **Create tab groups** in Chrome (right-click tab → Add to new group)
2. **Open TabRest popup** and click the group name
3. **All tabs in group unload** together

**Note**: Tab groups are a Chrome feature (Chrome 88+). TabRest integrates with existing groups.

## Statistics

Track your memory savings over time:

- **Tabs unloaded**: Total count since installation
- **Memory saved**: Estimated MB freed (calculated from tab memory usage)
- **View stats**: Open settings page
- **Reset stats**: Use reset button in settings

**Privacy**: All statistics stored locally. No data leaves your browser.

## Badge Indicator

Show the number of unloaded tabs on the extension icon:

- **Default**: Enabled
- **Display**: Small number badge on toolbar icon
- **Disable**: Turn off "Show badge count" in settings

## RAM Usage Display

View current system RAM usage in the popup header:

- **Display**: Shows RAM % alongside tab statistics
- **Warning**: Highlights when RAM exceeds your configured threshold
- **Tooltip**: Hover to see threshold comparison

## Scroll Position Restore

Automatically save and restore scroll position when tabs reload:

- **Default**: Enabled
- **Behavior**: Saves position before discard, restores on reload
- **Storage**: Local storage, auto-cleans old entries (max 100)

**Use case**: Continue reading long articles exactly where you left off.

## Auto-Unload Notifications

Get notified when tabs are automatically unloaded:

- **Default**: Disabled
- **Content**: Shows tab title, reason (Timer/RAM), and details
- **Silent**: Notifications don't play sound

**Use case**: Stay informed about what TabRest is doing in the background.

## Open Link Suspended

Open links directly in a suspended (discarded) state:

- **Access**: Right-click on any link → "Open Link in Suspended Tab"
- **Behavior**: Creates tab, waits for load, then immediately discards
- **Use case**: Save links for later without consuming memory
