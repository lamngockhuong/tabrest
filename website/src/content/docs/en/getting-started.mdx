# Getting Started with TabRest

TabRest is a browser extension that automatically unloads inactive tabs to free up memory. It works on **all Chromium-based browsers**: Chrome, Microsoft Edge, Brave, Opera, Vivaldi, and Arc.

## Installation

### From Browser Web Store

**Chrome users:**
1. Visit the [TabRest listing on Chrome Web Store](https://chrome.google.com/webstore) (coming soon)
2. Click **Add to Chrome**

**Edge users:**
1. Visit the [TabRest listing on Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons) (coming soon)
2. Click **Get**

**Brave, Opera, Vivaldi, Arc users:**
- Install directly from Chrome Web Store (these browsers support Chrome extensions)

3. Confirm the permission request
4. The TabRest icon will appear in your browser toolbar

### From Source (Developers)

1. Clone the repository:
   ```bash
   git clone https://github.com/lamngockhuong/tabrest.git
   cd tabrest
   ```

2. Open your browser and navigate to the extensions page:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`
   - Opera: `opera://extensions`
   - Vivaldi: `vivaldi://extensions`

3. Enable **Developer mode** (toggle in top-right corner)

4. Click **Load unpacked** and select the `tabrest` folder

5. The extension is now active

## First Steps

After installation, TabRest works immediately with default settings:

- **Auto-unload timer**: 30 minutes of inactivity
- **Memory threshold**: 80% RAM usage
- **Protected tabs**: Pinned tabs, tabs with audio, tabs with unsaved forms
- **Whitelist**: YouTube, Gmail, Google Docs (pre-configured)

### Quick Configuration

1. **Open settings**: Click the TabRest icon → ⚙️ Settings button
2. **Adjust timer**: Choose between 5 minutes to 4 hours
3. **Set memory threshold**: 60-95% RAM usage trigger
4. **Add to whitelist**: Protect specific sites from auto-unload

### Your First Manual Unload

Try unloading tabs manually to see how it works:

1. Open several tabs
2. Click the TabRest icon in the toolbar
3. Choose an action:
   - **Unload Current**: Unload the active tab
   - **Unload Others**: Unload all tabs except current
   - **Unload Left/Right**: Unload tabs on one side

Unloaded tabs stay visible in the tab bar but reload when you click them.

## What Happens When a Tab is Unloaded?

TabRest uses the browser's native `tabs.discard()` API. Unloaded tabs:

- ✅ Stay visible in the tab bar with their title and favicon
- ✅ Preserve scroll position and form data
- ✅ Reload instantly when clicked
- ✅ Free up memory while inactive
- ❌ Cannot run background scripts or play media

## Using Side Panel Mode

Want TabRest always visible instead of opening a popup?

1. Click TabRest icon → **⚙️ Settings** button
2. Scroll to **Toolbar Action** section
3. Toggle **Open in side panel** ON
4. The next click will open TabRest in the browser's sidebar
5. The panel stays open as you switch between tabs
6. Toggle off anytime to return to popup mode

**Note**: Side panel requires Chrome 114+.

## Auto-Opened Changelog

When TabRest updates with new features (minor or major versions), it automatically opens the changelog so you can see what's new. Patch updates (bug fixes) don't trigger the changelog to avoid unnecessary notifications.
