Texto de marketing longo usado no campo de descrição do Chrome Web Store.

```text
TabRest descarrega automaticamente abas inativas do navegador para liberar memória e manter seu computador funcionando sem problemas.

🎯 PERFEITO PARA:
• Usuários com muitas abas abertas
• Computadores com RAM limitada
• Quem quer uma experiência de navegação mais rápida

✨ RECURSOS PRINCIPAIS:

📌 Descarregamento Automático Inteligente
• Temporizador de inatividade configurável (5 minutos a 4 horas)
• Gatilho por limiar de memória (60-95% de uso de RAM)
• Limite de memória por aba (descarta abas usando >100MB-1GB de JS heap)
• Opção de descarregar ao iniciar - libera memória quando o navegador abre
• Ignorar quando offline - não descarregar abas quando a rede estiver indisponível

⌨️ Controles Manuais e Atalhos de Teclado
• Descarregar aba atual: Alt+Shift+D
• Descarregar outras abas: Alt+Shift+O
• Descarregar abas à direita: Alt+Shift+Right
• Descarregar abas à esquerda: Alt+Shift+Left
• Menu de contexto (clique com o botão direito) para acesso rápido

🛡️ Recursos de Proteção
• Lista branca de domínios para nunca descarregar
• Adiar abas individuais (30 min - 2 horas)
• Adiar domínios inteiros temporariamente
• Ignorar abas com áudio/vídeo em reprodução
• Ignorar abas com formulários não salvos
• Opção de proteger abas fixadas

📊 Estatísticas e Monitoramento
• Exibição de uso de RAM em tempo real no popup
• Rastrear total de abas descarregadas
• Rastrear total de memória economizada
• Indicador visual (💤) nos títulos das abas descartadas

🔧 Recursos Adicionais
• Suporte a grupos de abas - descarregar grupos inteiros
• Restauração de posição do YouTube - retome de onde parou
• Restauração de posição de rolagem quando as abas recarregam
• Notificações de descarregamento automático
• Fechar abas duplicadas - deduplicação com um clique no popup
• Busca de abas - filtro em tempo real por título ou URL
• Tooltip de estimativa de memória - passe o mouse para ver o detalhamento de memória por aba
• Toast de aviso de suspensão - aviso suave de 3 segundos antes do descarregamento automático
• Modo de painel lateral - abra a interface na barra lateral persistente do navegador em vez do popup (permanece visível enquanto você troca de aba)
• Importar/exportar configurações - faça backup e restaure listas brancas, listas negras e sessões para JSON na área de transferência
• Assistente de configuração interativo - integração guiada de 6 etapas na instalação (execute novamente a qualquer momento em Opções)
• Relatório de erros anônimo opcional (opt-in) - nos ajuda a corrigir bugs enquanto protege sua privacidade
• Suporte a múltiplos idiomas
• Abertura automática do registro de alterações - veja as novidades nas atualizações (apenas versões menores/maiores)

💡 COMO FUNCIONA:
TabRest usa a API nativa tabs.discard() do Chrome. Abas descartadas:
• Permanecem visíveis na sua barra de abas
• Preservam a posição de rolagem e os dados de formulário
• Recarregam instantaneamente ao clicar
• Liberam memória enquanto inativas

🔒 PRIVACIDADE:
• Nenhuma coleta de dados por padrão
• Relatório de erros anônimo opcional (desativado por padrão, totalmente transparente)
• Quando ativado: erros anonimizados, PII (URLs, e-mails, IPs) removido, máximo de 100 relatórios automáticos/dia, deduplicação + amostragem para reduzir ruído
• Todas as configurações armazenadas localmente no seu dispositivo
• Código aberto: https://github.com/lamngockhuong/tabrest

🌐 SITE: https://tabrest.ohnice.app

Feito com ❤️ para acumuladores de abas de todo o mundo.
```

## Data Usage / Uso de dados

Informações divulgadas na aba "Práticas de privacidade" do Chrome Web Store.

### Data Types Collected / Tipos de dados coletados

Marque **nenhuma** das categorias a seguir. O TabRest não coleta nenhuma delas.

| Categoria                                  | Coletado? | Motivo                                                                   |
| ------------------------------------------ | --------- | ------------------------------------------------------------------------ |
| Informações de identificação pessoal (PII) | Não       | Nenhum nome, e-mail ou endereço é coletado                               |
| Informações de saúde                       | Não       | Não aplicável                                                            |
| Informações financeiras/de pagamento       | Não       | Extensão gratuita, sem transações                                        |
| Informações de autenticação                | Não       | Nenhuma senha/PIN é coletada                                             |
| Comunicações pessoais                      | Não       | Nenhuma leitura de e-mail ou mensagens                                   |
| Localização                                | Não       | Nenhum IP ou GPS utilizado                                               |
| Histórico de navegação                     | Não       | URLs/títulos processados apenas localmente, nunca transmitidos           |
| Atividade do usuário                       | Não       | Nenhum registro de cliques/teclas; timestamps LRU armazenados localmente |
| Conteúdo de sites                          | Não       | Conteúdo da página não é lido nem transmitido                            |

> **Nota importante:** O TabRest acessa URLs/títulos das abas (via permissão `tabs`) e lê o estado de formulários (via `scripting`), mas tudo é processado localmente no dispositivo e **nunca sai da máquina do usuário**. Quando um usuário opta pelo relatório de erros anônimo, apenas stack traces e metadados sanitizados são enviados ao Sentry - sem URLs, PII ou conteúdo de página.

### Disclosures / Divulgações

Marque todas as 3:

- Não vendo nem transfiro dados dos usuários a terceiros (fora dos casos de uso aprovados - o Sentry atua como processador, não como venda de dados)
- Não uso nem transfiro dados dos usuários para fins não relacionados ao propósito único
- Não uso nem transfiro dados dos usuários para determinar capacidade de crédito ou para fins de empréstimo
