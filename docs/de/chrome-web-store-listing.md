## Full Description / Vollständige Beschreibung

Ausführlicher Marketingtext für das Beschreibungsfeld im Chrome Web Store.

```text
TabRest entlädt inaktive Browser-Tabs automatisch, um Arbeitsspeicher freizugeben und Ihren Computer flüssig laufen zu lassen.

🎯 IDEAL FÜR:
• Nutzer mit vielen geöffneten Tabs
• Computer mit begrenztem RAM
• Alle, die ein schnelleres Surferlebnis wünschen

✨ HAUPTFUNKTIONEN:

📌 Intelligentes automatisches Entladen
• Konfigurierbarer Inaktivitäts-Timer (5 Minuten bis 4 Stunden)
• Speicherschwelle als Auslöser (60-95 % RAM-Auslastung)
• Tab-spezifisches Speicherlimit (Tabs entladen, die mehr als 100 MB-1 GB JS-Heap verbrauchen)
• Option zum Entladen beim Start, um beim Öffnen des Browsers Speicher freizugeben
• Überspringen bei fehlender Verbindung - Tabs nicht entladen, wenn kein Netzwerk verfügbar

⌨️ Manuelle Steuerung und Tastenkürzel
• Aktuellen Tab entladen: Alt+Shift+D
• Andere Tabs entladen: Alt+Shift+O
• Tabs rechts davon entladen: Alt+Shift+Right
• Tabs links davon entladen: Alt+Shift+Left
• Kontextmenü per Rechtsklick für schnellen Zugriff

🛡️ Schutzfunktionen
• Whitelist für Domains, die niemals entladen werden sollen
• Einzelne Tabs snoozen (30 Min. bis 2 Stunden)
• Ganze Domains vorübergehend snoozen
• Tabs mit laufender Audio-/Videowiedergabe überspringen
• Tabs mit nicht gespeicherten Formulardaten überspringen
• Option zum Schutz angehefteter Tabs

📊 Statistiken und Überwachung
• Live-RAM-Anzeige im Popup
• Gesamtzahl der entladenen Tabs verfolgen
• Gesamten eingesparten Speicher verfolgen
• Visueller Indikator (💤) in Titeln verworfener Tabs

🔧 Weitere Funktionen
• Tab-Gruppen-Unterstützung - ganze Gruppen entladen
• YouTube-Positionswiederherstellung - weiterschauen, wo Sie aufgehört haben
• Scroll-Position beim Neuladen von Tabs wiederherstellen
• Benachrichtigungen beim automatischen Entladen
• Doppelte Tabs schließen - Ein-Klick-Deduplizierung im Popup
• Tab-Suche - Echtzeit-Filter nach Titel oder URL
• Speicherschätzungs-Tooltip - mit der Maus über einen Tab fahren, um die Speicheraufschlüsselung je Tab anzuzeigen
• Warnhinweis vor dem Entladen - sanfte 3-Sekunden-Warnung vor dem automatischen Entladen
• Seitenleistenmodus - Benutzeroberfläche in einer dauerhaften Browser-Seitenleiste öffnen statt als Popup (bleibt beim Tab-Wechsel sichtbar)
• Einstellungen importieren/exportieren - Whitelists, Blacklists und Sitzungen als JSON in die Zwischenablage sichern und wiederherstellen
• Interaktiver Einrichtungsassistent - geführtes 6-Schritte-Onboarding bei der Installation (jederzeit über die Optionsseite erneut starten)
• Optionale anonyme Fehlerberichte (opt-in) - hilft uns, Fehler zu beheben, ohne Ihre Privatsphäre zu gefährden
• Mehrsprachige Unterstützung
• Automatisches Öffnen des Änderungsprotokolls - Neuigkeiten bei Updates anzeigen (nur bei Minor- und Major-Releases)

💡 SO FUNKTIONIERT ES:
TabRest verwendet die native tabs.discard()-API von Chrome. Verworfene Tabs:
• Bleiben in Ihrer Tab-Leiste sichtbar
• Behalten die Scroll-Position und Formulardaten
• Laden sofort neu, wenn Sie darauf klicken
• Geben Speicher frei, während sie inaktiv sind

🔒 DATENSCHUTZ:
• Standardmäßig keine Datenerfassung
• Optionale anonyme Fehlerberichte (standardmäßig deaktiviert, vollständig transparent)
• Wenn aktiviert: Fehler anonymisiert, personenbezogene Daten entfernt, max. 100/Tag, 30 Tage Aufbewahrung
• Alle Einstellungen lokal auf Ihrem Gerät gespeichert
• Open Source: https://github.com/lamngockhuong/tabrest

🌐 WEBSITE: https://tabrest.ohnice.app

Mit ❤️ für Tab-Sammler überall auf der Welt.
```

## Data Usage / Datennutzung

Informationen, die im Chrome Web Store auf der Registerkarte "Datenschutzpraktiken" offengelegt werden.

### Data Types Collected / Erfasste Datentypen

Keiner der folgenden Kategorien ankreuzen. TabRest erfasst keine davon.

| Kategorie                         | Erfasst? | Grund                                                                             |
| --------------------------------- | -------- | --------------------------------------------------------------------------------- |
| Personenbezogene Daten (PII)      | Nein     | Kein Name, keine E-Mail-Adresse oder Anschrift wird erfasst                       |
| Gesundheitsinformationen          | Nein     | Nicht zutreffend                                                                  |
| Finanz- und Zahlungsinformationen | Nein     | Kostenlose Erweiterung, keine Transaktionen                                       |
| Authentifizierungsinformationen   | Nein     | Kein Passwort oder PIN wird erfasst                                               |
| Persönliche Kommunikation         | Nein     | Keine E-Mails oder Nachrichten werden gelesen                                     |
| Standort                          | Nein     | Keine IP- oder GPS-Nutzung                                                        |
| Webverlauf                        | Nein     | URLs/Titel werden nur lokal verarbeitet und niemals übertragen                    |
| Nutzeraktivität                   | Nein     | Kein Klick- oder Tastenanschlag-Logging; LRU-Zeitstempel werden lokal gespeichert |
| Website-Inhalt                    | Nein     | Seiteninhalt wird nicht gelesen oder übertragen                                   |

> **Wichtiger Hinweis:** TabRest greift auf Tab-URLs und -Titel zu (über die Berechtigung `tabs`) und liest den Formularzustand (über `scripting`), aber alles wird lokal auf dem Gerät verarbeitet und **verlässt niemals das Gerät des Nutzers**. Wenn ein Nutzer sich für anonyme Fehlerberichte entscheidet, werden nur bereinigte Stack-Traces und Metadaten an Sentry gesendet - keine URLs, personenbezogenen Daten oder Seiteninhalt.

### Disclosures / Offenlegungen

Alle 3 ankreuzen:

- Ich verkaufe oder übertrage keine Nutzerdaten an Dritte (außerhalb genehmigter Anwendungsfälle - Sentry fungiert als Auftragsverarbeiter, nicht als Datenverkauf)
- Ich verwende oder übertrage keine Nutzerdaten für Zwecke, die nichts mit dem einzigen Zweck der Erweiterung zu tun haben
- Ich verwende oder übertrage keine Nutzerdaten zur Bestimmung der Kreditwürdigkeit oder für Kreditvergabezwecke
