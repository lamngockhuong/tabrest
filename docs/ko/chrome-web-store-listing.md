# Chrome Web Store Listing - TabRest (한국어)

## What's New / 새로운 기능

각 릴리스마다 아래 내용을 Chrome Web Store "새로운 기능" 필드에 붙여 넣으십시오.

```
신규: 인터랙티브 설정 마법사

처음 설치하면 6단계 안내 마법사가 열려 가장 유용한 설정을 안내해 드립니다 - 자동 언로드 타이머, 허용 목록 제안, 전원 모드, 알림 등. 각 단계는 건너뛸 수 있으며, 건너뛰어도 기존 기본값이 그대로 유지됩니다.

이미 설치하셨나요? 옵션 > 하단 > "설정 마법사 다시 실행"에서 언제든지 마법사를 다시 실행할 수 있습니다.

기타 개선 사항:
- 부드러운 슬라이드 전환 효과가 적용된 설치/환영 화면 개선
- 접근성을 위한 모션 감소 지원
- TabRest를 더 빠르게 찾을 수 있도록 도구 모음에 고정 안내 추가
- 옵션 페이지 하단 디자인 개선
- 다국어 마법사 지원 (영어 및 베트남어)

개인 정보 보호:
모든 설정은 기기에 로컬로 저장됩니다. 선택적 익명 오류 보고는 옵트인 방식이며 기본적으로 비활성화되어 있습니다.
```

## Short Description / 짧은 설명

최대 132자. 스토어 목록 태그라인으로 사용됩니다.

```text
탭을 쉬게 하고 메모리를 확보하세요 - 비활성 탭을 자동으로 언로드하여 Chrome을 빠르고 원활하게 유지합니다.
```

## Full Description / 전체 설명

Chrome Web Store 설명 필드에 사용되는 장문 마케팅 문구입니다.

```text
TabRest는 비활성 브라우저 탭을 자동으로 언로드하여 메모리를 확보하고 컴퓨터를 원활하게 유지합니다.

🎯 이런 분께 딱 맞습니다:
• 탭을 많이 열어 두는 사용자
• RAM이 제한된 컴퓨터 사용자
• 더 빠른 브라우징 환경을 원하는 모든 분

✨ 주요 기능:

📌 스마트 자동 언로드
• 설정 가능한 비활성 타이머 (5분 - 4시간)
• 메모리 임계값 트리거 (RAM 사용량 60-95%)
• 탭별 메모리 제한 (JS heap >100MB-1GB 사용 탭 언로드)
• 브라우저 시작 시 언로드로 메모리 확보
• 오프라인 시 건너뛰기 - 네트워크 미사용 중에는 탭을 언로드하지 않습니다

⌨️ 수동 제어 및 키보드 단축키
• 현재 탭 언로드: Alt+Shift+D
• 다른 탭 언로드: Alt+Shift+O
• 오른쪽 탭 언로드: Alt+Shift+Right
• 왼쪽 탭 언로드: Alt+Shift+Left
• 빠른 접근을 위한 컨텍스트 메뉴

🛡️ 보호 기능
• 도메인을 허용 목록에 추가하여 언로드 방지
• 개별 탭 스누즈 (30분 - 2시간)
• 도메인 전체 일시적 스누즈
• 오디오/동영상 재생 중인 탭 건너뛰기
• 저장되지 않은 양식 데이터가 있는 탭 건너뛰기
• 고정된 탭 보호 옵션

📊 통계 및 모니터링
• 팝업에서 실시간 RAM 사용량 표시
• 총 언로드된 탭 수 추적
• 총 절약된 메모리 추적
• 삭제된 탭 제목의 시각적 표시 (💤)

🔧 추가 기능
• 탭 그룹 지원 - 그룹 전체 언로드
• YouTube 재생 위치 복원 - 중단한 지점부터 재개
• 탭 새로 고침 시 스크롤 위치 복원
• 자동 언로드 알림
• 중복 탭 닫기 - 팝업에서 원클릭 중복 제거
• 탭 검색 - 제목 또는 URL로 실시간 필터링
• 메모리 예상 툴팁 - 마우스를 올리면 탭별 메모리 분석 표시
• 언로드 경고 토스트 - 자동 언로드 3초 전 부드러운 경고
• 측면 패널 모드 - 팝업 대신 브라우저 사이드바에서 UI 열기 (탭 전환 시에도 계속 표시)
• 설정 가져오기/내보내기 - 허용 목록, 차단 목록, 세션을 클립보드 JSON으로 백업 및 복원
• 인터랙티브 설정 마법사 - 설치 시 6단계 온보딩 안내 (옵션 페이지에서 언제든지 재실행 가능)
• 선택적 익명 오류 보고 (옵트인) - 개인 정보를 보호하면서 버그 수정에 도움
• 다국어 지원
• 변경 로그 자동 열기 - 업데이트 시 새 내용 확인 (부/주요 릴리스에만 해당)

💡 작동 원리:
TabRest는 Chrome의 기본 tabs.discard() API를 사용합니다. 삭제된 탭은:
• 탭 표시줄에 계속 표시됩니다
• 스크롤 위치와 양식 데이터가 유지됩니다
• 클릭 시 즉시 다시 로드됩니다
• 비활성 상태에서 메모리를 확보합니다

🔒 개인 정보 보호:
• 기본적으로 데이터를 수집하지 않습니다
• 선택적 익명 오류 보고 (기본 비활성화, 완전 투명)
• 활성화 시: 오류 익명 처리, PII (URL, 이메일, IP) 제거, 자동 보고 하루 최대 100건, 중복 제거 + 샘플링으로 노이즈 감소
• 모든 설정은 기기에 로컬로 저장됩니다
• 오픈 소스: https://github.com/lamngockhuong/tabrest

🌐 웹사이트: https://tabrest.ohnice.app

탭을 쌓아 두는 모든 분을 위해 ❤️로 만들었습니다.
```

## Data Usage / 데이터 사용

Chrome Web Store "개인 정보 보호 관행" 탭에 공개하는 정보입니다.

### Data Types Collected / 수집하는 데이터 유형

아래 카테고리 중 **어느 것도** 선택하지 마십시오. TabRest는 이 중 어떤 데이터도 수집하지 않습니다.

| 카테고리             | 수집 여부 | 이유                                                                |
| -------------------- | --------- | ------------------------------------------------------------------- |
| 개인 식별 정보 (PII) | 아니요    | 이름, 이메일, 주소를 수집하지 않습니다                              |
| 건강 정보            | 아니요    | 해당 없음                                                           |
| 금융/결제 정보       | 아니요    | 무료 확장 프로그램이며 거래가 없습니다                              |
| 인증 정보            | 아니요    | 비밀번호/PIN을 수집하지 않습니다                                    |
| 개인 통신 내용       | 아니요    | 이메일이나 메시지를 읽지 않습니다                                   |
| 위치 정보            | 아니요    | IP 또는 GPS를 사용하지 않습니다                                     |
| 웹 방문 기록         | 아니요    | URL/제목은 로컬에서만 처리되며 전송되지 않습니다                    |
| 사용자 활동          | 아니요    | 클릭/키 입력을 기록하지 않으며 LRU 타임스탬프는 로컬에만 저장됩니다 |
| 웹사이트 콘텐츠      | 아니요    | 페이지 콘텐츠를 읽거나 전송하지 않습니다                            |

> **중요 사항:** TabRest는 `tabs` 권한을 통해 탭 URL/제목에 접근하고 `scripting`을 통해 양식 상태를 읽지만, 모든 처리는 기기에서 로컬로 이루어지며 **사용자 기기 밖으로 전송되지 않습니다**. 사용자가 익명 오류 보고에 동의한 경우, Sentry로는 정제된 스택 트레이스와 메타데이터만 전송됩니다 - URL, PII, 페이지 콘텐츠는 포함되지 않습니다.

### Disclosures / 공개 사항

아래 3가지 모두 체크하십시오:

- 승인된 사용 사례 외에는 사용자 데이터를 제3자에게 판매하거나 이전하지 않습니다 (Sentry는 데이터 판매가 아닌 처리자로서 역할합니다)
- 단일 목적과 관련 없는 용도로 사용자 데이터를 사용하거나 이전하지 않습니다
- 신용 평가 또는 대출 목적으로 사용자 데이터를 사용하거나 이전하지 않습니다
