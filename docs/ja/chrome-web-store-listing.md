# Chrome Web Store Listing - TabRest (日本語)

## What's New / 新着情報

各リリース時に、以下の内容を Chrome Web Store の「新着情報」欄に貼り付けてください。

```
新機能: インタラクティブなセットアップウィザード

初回インストール時に、6 ステップのわかりやすいウィザードが起動します。自動アンロードタイマー、ホワイトリストの候補、電源モード、通知など、便利な設定を順を追って案内します。各ステップはスキップ可能で、スキップした場合も既存のデフォルト値はそのまま保持されます。

すでにインストール済みの方は、オプションページ > フッター > 「セットアップを再実行」からいつでもウィザードを再実行できます。

その他の改善点:
- スムーズなスライドトランジションによるインストール/ウェルカム画面のリニューアル
- アクセシビリティのための動きの軽減 (reduced-motion) サポート
- TabRest をすぐに見つけられるツールバーへのピン留めリマインダー
- オプションページのフッターを整備
- 多言語対応ウィザード (英語とベトナム語)

プライバシー:
すべての設定はお使いのデバイスにローカルで保存されます。任意の匿名エラーレポートはオプトイン制です (デフォルトは無効)。
```

## Short Description / 短い説明

132 文字以内。ストアリストのタグラインとして使用されます。

```text
タブを休ませ、メモリを解放 - 非アクティブなタブを自動アンロードして Chrome を高速かつ快適に保ちます。
```

## Full Description / 詳細説明

Chrome Web Store の説明欄に使用するロング形式のマーケティングコピーです。

```text
TabRest は非アクティブなブラウザタブを自動でアンロードしてメモリを解放し、コンピューターをスムーズに動作させます。

🎯 こんな方に最適:
• タブを多く開いている方
• RAM が限られたコンピューターをお使いの方
• より快適なブラウジング体験を求める方

✨ 主な機能:

📌 スマート自動アンロード
• 非アクティブタイマーの設定 (5 分から 4 時間)
• メモリしきい値トリガー (RAM 使用率 60 から 95%)
• タブごとのメモリ上限 (JS heap が 100MB から 1GB を超えたタブをアンロード)
• ブラウザ起動時にアンロードするオプション
• オフライン時はスキップ - ネットワーク未接続時はタブをアンロードしない

⌨️ 手動コントロールとキーボードショートカット
• 現在のタブをアンロード: Alt+Shift+D
• 他のタブをアンロード: Alt+Shift+O
• 右側のタブをアンロード: Alt+Shift+Right
• 左側のタブをアンロード: Alt+Shift+Left
• コンテキストメニューからすばやくアクセス

🛡️ 保護機能
• ドメインをホワイトリストに追加してアンロードを防止
• 個別タブのスヌーズ (30 分から 2 時間)
• ドメイン全体を一時的にスヌーズ
• 音声/動画を再生中のタブをスキップ
• 未保存のフォームがあるタブをスキップ
• ピン留めタブを保護するオプション

📊 統計とモニタリング
• ポップアップでリアルタイム RAM 使用量を表示
• アンロードしたタブの累計数を記録
• 節約したメモリの累計量を記録
• 破棄済みタブのタイトルにビジュアルインジケーター (💤) を表示

🔧 追加機能
• タブグループのサポート - グループ全体をアンロード
• YouTube 再生位置の復元 - 中断した位置から再開
• タブ再読み込み時のスクロール位置の復元
• 自動アンロード通知
• 重複タブを閉じる - ポップアップからワンクリックで重複を削除
• タブ検索 - タイトルまたは URL でリアルタイムにフィルタリング
• メモリ推定ツールチップ - ホバーでタブごとのメモリ内訳を表示
• 一時停止警告トースト - 自動アンロード前に 3 秒間のやさしい警告を表示
• サイドパネルモード - ポップアップの代わりにブラウザの常設サイドバーに UI を表示 (タブを切り替えても表示されたまま)
• 設定のインポート/エクスポート - ホワイトリスト、ブラックリスト、セッションをクリップボードの JSON 形式でバックアップおよび復元
• インタラクティブなセットアップウィザード - インストール時に 6 ステップのガイド付きオンボーディング (オプションページからいつでも再実行可能)
• 任意の匿名エラーレポート (オプトイン制) - プライバシーを守りながらバグ修正に貢献
• 多言語サポート
• 変更履歴の自動表示 - アップデート時に新着情報を表示 (マイナー/メジャーリリースのみ)

💡 仕組み:
TabRest は Chrome のネイティブ API である `chrome.tabs.discard()` を使用します。破棄済みタブは:
• タブバーに引き続き表示されます
• スクロール位置とフォームデータを保持します
• クリックするとすぐに再読み込みされます
• 非アクティブ中はメモリを解放します

🔒 プライバシー:
• デフォルトではデータを収集しません
• 任意の匿名エラーレポート (デフォルトは無効、完全に透明)
• 有効時: エラーは匿名化、PII (URL・メール・IP) は除去、自動レポート 1 日最大 100 件、重複排除 + サンプリングでノイズ軽減
• すべての設定はデバイスにローカルで保存されます
• オープンソース: https://github.com/lamngockhuong/tabrest

🌐 ウェブサイト: https://tabrest.ohnice.app

タブをため込む全ての方へ、❤️ を込めて。
```

## Data Usage / データ使用状況

Chrome Web Store の「プライバシーの取り組み」タブに開示する情報です。

### Data Types Collected / 収集するデータの種類

以下のカテゴリはいずれも**チェックしない**でください。TabRest はこれらのデータを一切収集しません。

| カテゴリ                   | 収集する? | 理由                                                                      |
| -------------------------- | --------- | ------------------------------------------------------------------------- |
| 個人を特定できる情報 (PII) | いいえ    | 氏名、メールアドレス、住所は収集しない                                    |
| 健康情報                   | いいえ    | 該当なし                                                                  |
| 金融/決済情報              | いいえ    | 無料の拡張機能のため取引なし                                              |
| 認証情報                   | いいえ    | パスワード/PIN は収集しない                                               |
| 個人通信                   | いいえ    | メールやメッセージの読み取りなし                                          |
| 位置情報                   | いいえ    | IP や GPS は使用しない                                                    |
| ウェブ閲覧履歴             | いいえ    | URL/タイトルはローカルでのみ処理し、外部に送信しない                      |
| ユーザーアクティビティ     | いいえ    | クリック/キーストロークのログ記録なし。LRU タイムスタンプはローカルに保存 |
| ウェブサイトのコンテンツ   | いいえ    | ページコンテンツの読み取りや送信なし                                      |

> **重要な注意:** TabRest は (`tabs` 権限を通じて) タブの URL/タイトルにアクセスし、(`scripting` を通じて) フォームの状態を読み取りますが、すべてデバイス上でローカルに処理され、**ユーザーのデバイス外に送信されることはありません**。ユーザーが匿名エラーレポートをオプトインした場合、サニタイズされたスタックトレースとメタデータのみが Sentry に送信されます。URL、PII、ページコンテンツは含まれません。

### Disclosures / 開示事項

以下の 3 項目すべてにチェックを入れてください:

- 承認された利用目的以外でユーザーデータを第三者に販売または譲渡しない (Sentry はデータ処理者として機能し、データの販売は行わない)
- 単一の目的と無関係な目的でユーザーデータを使用または譲渡しない
