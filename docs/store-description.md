## Detailed Description

TabRest - Rest Your Tabs, Free Your Memory

Features:

- Auto-unload inactive tabs after customizable time (15min to 4hrs)
- Unload tabs when memory usage exceeds threshold (60-90%)
- Unload all tabs on browser startup
- Visual indicator - customizable prefix (💤) on discarded tab titles
- Keyboard shortcuts for quick actions
- Right-click context menu integration
- Tab groups support
- Whitelist your favorite sites
- Track memory saved statistics
- Side panel mode - open UI in sidebar instead of popup
- Import/export settings - backup and restore your configuration

Keyboard Shortcuts:

- Alt+Shift+D - Unload current tab
- Alt+Shift+O - Unload other tabs
- Alt+Shift+Right - Unload tabs to the right
- Alt+Shift+Left - Unload tabs to the left

How it works:
TabRest uses Chrome's native tab discard feature to free up memory from inactive tabs while preserving your browsing state. When you click on a resting tab, it instantly reloads.

Privacy:

- No data collection
- No external servers
- All settings stored locally
- Open source

Perfect for users with many tabs open who want to keep their browser fast and responsive!
