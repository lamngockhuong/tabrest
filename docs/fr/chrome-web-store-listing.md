## What's New / Nouveautés

Collez le contenu ci-dessous dans le champ "Nouveautés" du Chrome Web Store à chaque version.

```
NOUVEAU : Assistant de configuration interactif

La première installation ouvre désormais un assistant convivial en 6 étapes qui vous guide à travers les paramètres les plus utiles - minuteur de déchargement automatique, suggestions de liste blanche, mode d'alimentation et notifications. Tous les paramètres sont stockés localement sur votre appareil. Le rapport d'erreur anonyme facultatif reste en option d'adhésion (désactivé par défaut).
```

## Full Description / Description complète

Texte marketing long utilisé dans le champ de description du Chrome Web Store.

```text
TabRest décharge automatiquement les onglets de navigateur inactifs pour libérer de la mémoire et maintenir votre ordinateur en pleine forme.

🎯 IDÉAL POUR :
• Les utilisateurs ayant de nombreux onglets ouverts
• Les ordinateurs avec peu de RAM
• Tous ceux qui souhaitent une navigation plus rapide

✨ FONCTIONNALITÉS CLÉS :

📌 Déchargement automatique intelligent
• Minuteur d'inactivité configurable (5 minutes à 4 heures)
• Déclenchement par seuil de mémoire (60-95% d'utilisation RAM)
• Limite de mémoire par onglet (décharger les onglets utilisant >100 Mo-1 Go de JS heap)
• Option de déchargement au démarrage pour libérer la mémoire à l'ouverture du navigateur
• Ignorer hors connexion - ne pas décharger les onglets en l'absence de réseau

⌨️ Contrôles manuels et raccourcis clavier
• Décharger l'onglet actuel : Alt+Shift+D
• Décharger les autres onglets : Alt+Shift+O
• Décharger les onglets à droite : Alt+Shift+Right
• Décharger les onglets à gauche : Alt+Shift+Left
• Menu contextuel au clic droit pour un accès rapide

🛡️ Fonctionnalités de protection
• Liste blanche de domaines à ne jamais décharger
• Reporter un onglet individuellement (30 min - 2 heures)
• Reporter l'ensemble d'un domaine temporairement
• Ignorer les onglets avec de l'audio/vidéo en lecture
• Ignorer les onglets avec des formulaires non enregistrés
• Option de protection des onglets épinglés

📊 Statistiques et surveillance
• Affichage en direct de l'utilisation RAM dans le popup
• Suivi du nombre total d'onglets déchargés
• Suivi de la mémoire totale économisée
• Indicateur visuel (💤) dans les titres des onglets ignorés

🔧 Fonctionnalités supplémentaires
• Prise en charge des groupes d'onglets - décharger des groupes entiers
• Restauration de la position YouTube - reprenez là où vous vous étiez arrêté
• Restauration de la position de défilement au rechargement des onglets
• Notifications de déchargement automatique
• Fermeture des onglets en double - déduplication en un clic dans le popup
• Recherche d'onglets - filtre en direct par titre ou URL
• Info-bulle d'estimation de la mémoire - survolez pour voir la répartition de la mémoire par onglet
• Toast d'avertissement de suspension - avertissement discret de 3 secondes avant le déchargement automatique
• Mode panneau latéral - ouvrez l'interface dans la barre latérale persistante du navigateur plutôt qu'en popup (reste visible lorsque vous changez d'onglets)
• Import/export des paramètres - sauvegardez et restaurez les listes blanches, listes noires et sessions en JSON dans le presse-papier
• Assistant de configuration interactif - intégration guidée en 6 étapes à l'installation (relancez à tout moment depuis les Options)
• Rapport d'erreur anonyme facultatif (opt-in) - nous aide à corriger les bugs tout en protégeant votre vie privée
• Prise en charge multilingue
• Ouverture automatique du journal des modifications - consultez les nouveautés lors des mises à jour (versions mineures/majeures uniquement)

💡 COMMENT CA FONCTIONNE :
TabRest utilise l'API native Chrome tabs.discard(). Les onglets ignorés :
• Restent visibles dans votre barre d'onglets
• Conservent la position de défilement et les données de formulaire
• Se rechargent instantanément au clic
• Libèrent de la mémoire lorsqu'ils sont inactifs

🔒 CONFIDENTIALITÉ :
• Aucune collecte de données par défaut
• Rapport d'erreur anonyme facultatif (désactivé par défaut, entièrement transparent)
• Lorsqu'il est activé : erreurs anonymisées, PII supprimées, max 100/jour, conservation 30 jours
• Tous les paramètres stockés localement sur votre appareil
• Open source : https://github.com/lamngockhuong/tabrest

🌐 SITE WEB : https://tabrest.ohnice.app

Fait avec ❤️ pour les collectionneurs d'onglets du monde entier.
```

## Data Usage / Utilisation des données

Informations divulguées dans l'onglet "Pratiques de confidentialité" du Chrome Web Store.

### Data Types Collected / Types de données collectées

Cochez **aucune** des catégories suivantes. TabRest n'en collecte aucune.

| Catégorie                                        | Collectée ? | Raison                                                                                            |
| ------------------------------------------------ | ----------- | ------------------------------------------------------------------------------------------------- |
| Informations personnellement identifiables (PII) | Non         | Aucun nom, e-mail ou adresse collecté                                                             |
| Informations de santé                            | Non         | Sans objet                                                                                        |
| Informations financières / de paiement           | Non         | Extension gratuite, aucune transaction                                                            |
| Informations d'authentification                  | Non         | Aucun mot de passe / code PIN collecté                                                            |
| Communications personnelles                      | Non         | Aucune lecture d'e-mail ou de message                                                             |
| Localisation                                     | Non         | Aucune adresse IP ou GPS utilisée                                                                 |
| Historique de navigation                         | Non         | Les URL/titres sont traités localement uniquement, jamais transmis                                |
| Activité de l'utilisateur                        | Non         | Aucun enregistrement de clics ou de frappes clavier ; les horodatages LRU sont stockés localement |
| Contenu des sites web                            | Non         | Le contenu des pages n'est ni lu ni transmis                                                      |

> **Note importante :** TabRest accède aux URL/titres des onglets (via la permission `tabs`) et lit l'état des formulaires (via `scripting`), mais tout est traité localement sur l'appareil et **ne quitte jamais la machine de l'utilisateur**. Lorsqu'un utilisateur accepte le rapport d'erreur anonyme, seules des traces de pile et des métadonnées nettoyées sont envoyées à Sentry - aucune URL, PII ou contenu de page n'est inclus.

### Disclosures / Divulgations

Cochez les 3 cases :

- Je ne vends pas ni ne transfère les données des utilisateurs à des tiers (en dehors des cas d'utilisation approuvés - Sentry agit en tant que sous-traitant, non comme acheteur de données)
- Je n'utilise pas ni ne transfère les données des utilisateurs à des fins sans rapport avec l'objectif principal
- Je n'utilise pas ni ne transfère les données des utilisateurs pour évaluer la solvabilité ou à des fins de prêt
