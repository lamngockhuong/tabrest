## Full Description / 完整描述

用于 Chrome Web Store 描述字段的长篇营销文案。

```text
TabRest 自动释放不活跃的浏览器标签页内存，让您的电脑保持流畅运行。

🎯 适合以下用户：
• 同时打开大量标签页的用户
• RAM 有限的电脑
• 想要更快浏览体验的用户

✨ 核心功能：

📌 智能自动释放
• 可配置闲置计时器（5 分钟至 4 小时）
• 内存阈值触发（RAM 使用率 60-95%）
• 单标签页内存限制（释放 JS heap 占用 >100MB-1GB 的标签页）
• 启动时释放选项，在浏览器打开时释放内存
• 离线时跳过 - 网络不可用时不释放标签页

⌨️ 手动控制与键盘快捷键
• 释放当前标签页：Alt+Shift+D
• 释放其他标签页：Alt+Shift+O
• 释放右侧标签页：Alt+Shift+Right
• 释放左侧标签页：Alt+Shift+Left
• 右键菜单快速访问

🛡️ 保护功能
• 白名单域名，永不释放
• 暂停单个标签页（30 分钟 - 2 小时）
• 临时暂停整个域名
• 跳过正在播放音频/视频的标签页
• 跳过含有未保存表单的标签页
• 保护已固定标签页选项

📊 统计与监控
• 弹窗中实时显示 RAM 使用情况
• 追踪已释放标签页总数
• 追踪已节省内存总量
• 已丢弃标签页标题上显示视觉指示符（💤）

🔧 附加功能
• 标签组支持 - 释放整个标签组
• YouTube 播放位置恢复 - 从上次离开处继续播放
• 标签页重新加载时恢复滚动位置
• 自动释放通知
• 关闭重复标签页 - 在弹窗中一键去重
• 标签页搜索 - 按标题或 URL 实时筛选
• 内存估算提示 - 悬停查看每个标签页的内存明细
• 暂停警告提示 - 自动释放前 3 秒温和提醒
• 侧边栏模式 - 在浏览器持久侧边栏中打开界面，切换标签页时保持可见
• 导入/导出设置 - 将白名单、黑名单和会话备份并恢复至剪贴板 JSON
• 交互式设置向导 - 安装时引导完成 6 步配置（可随时从选项页面重新运行）
• 可选匿名错误报告（选择加入）- 帮助我们修复问题，同时保护您的隐私
• 多语言支持
• 自动打开更新日志 - 更新时查看最新动态（仅限次要/主要版本）

💡 工作原理：
TabRest 使用 Chrome 原生 `chrome.tabs.discard()` API。已丢弃的标签页：
• 仍显示在标签栏中
• 保留滚动位置和表单数据
• 点击时立即重新加载
• 不活跃时释放内存

🔒 隐私：
• 默认不收集任何数据
• 可选匿名错误报告（默认禁用，完全透明）
• 启用时：错误匿名化处理，PII (URL、邮箱、IP) 已脱敏，自动报告每日最多 100 条，去重 + 采样减少噪音
• 所有设置均存储在您的设备本地
• 开源：https://github.com/lamngockhuong/tabrest

🌐 网站：https://tabrest.ohnice.app

用 ❤️ 为每一位"标签页收藏家"打造。
```

## Data Usage / 数据使用情况

在 Chrome Web Store "隐私惯例" 标签页中披露的信息。

### Data Types Collected / 收集的数据类型

以下所有类别均**不勾选**。TabRest 不收集其中任何数据。

| 类别                | 是否收集？ | 原因                                    |
| ------------------- | ---------- | --------------------------------------- |
| 个人身份信息（PII） | 否         | 不收集姓名、电子邮件或地址              |
| 健康信息            | 否         | 不适用                                  |
| 财务/支付信息       | 否         | 免费扩展，无任何交易                    |
| 身份验证信息        | 否         | 不收集密码/PIN                          |
| 个人通信内容        | 否         | 不读取电子邮件或消息                    |
| 位置信息            | 否         | 不使用 IP 或 GPS                        |
| 网络历史记录        | 否         | URL/标题仅在本地处理，从不传输          |
| 用户活动            | 否         | 不记录点击/按键；LRU 时间戳仅存储在本地 |
| 网站内容            | 否         | 不读取或传输页面内容                    |

> **重要说明：** TabRest 通过 `tabs` 权限访问标签页 URL/标题，并通过 `scripting` 读取表单状态，但所有内容均在设备本地处理，**从不离开用户的设备**。当用户选择加入匿名错误报告时，仅将经过脱敏处理的堆栈跟踪和元数据发送至 Sentry - 不包含 URL、PII 或页面内容。

### Disclosures / 披露声明

以下 3 项全部勾选：

- 我不会将用户数据出售或转让给第三方（经批准的使用场景除外 - Sentry 作为数据处理方，并非数据出售）
- 我不会将用户数据用于与单一用途无关的目的
- 我不会将用户数据用于信用评估或贷款目的
