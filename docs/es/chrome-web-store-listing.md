## Full Description / Descripción completa

Texto de marketing largo utilizado en el campo de descripción de Chrome Web Store.

```text
TabRest descarga automáticamente las pestañas del navegador inactivas para liberar memoria y mantener su computadora funcionando sin problemas.

🎯 IDEAL PARA:
• Usuarios con muchas pestañas abiertas
• Computadoras con RAM limitada
• Quienes desean una experiencia de navegación más rápida

✨ CARACTERÍSTICAS PRINCIPALES:

📌 Descarga automática inteligente
• Temporizador de inactividad configurable (5 minutos a 4 horas)
• Activación por umbral de memoria (60-95% de uso de RAM)
• Límite de memoria por pestaña (descarta pestañas que usan >100MB-1GB de JS heap)
• Opción de descargar al inicio para liberar memoria al abrir el navegador
• Omitir cuando está sin conexión - no descarga pestañas cuando la red no está disponible

⌨️ Controles manuales y atajos de teclado
• Descargar pestaña actual: Alt+Shift+D
• Descargar otras pestañas: Alt+Shift+O
• Descargar pestañas a la derecha: Alt+Shift+Right
• Descargar pestañas a la izquierda: Alt+Shift+Left
• Menú contextual para acceso rápido

🛡️ Funciones de protección
• Lista blanca de dominios para nunca descargar
• Posponer pestañas individuales (30 min - 2 horas)
• Posponer dominios completos temporalmente
• Omitir pestañas con audio/video en reproducción
• Omitir pestañas con formularios sin guardar
• Opción para proteger pestañas fijadas

📊 Estadísticas y monitoreo
• Visualización en vivo del uso de RAM en la ventana emergente
• Seguimiento del total de pestañas descargadas
• Seguimiento del total de memoria ahorrada
• Indicador visual (💤) en los títulos de pestañas descartadas

🔧 Funciones adicionales
• Soporte para grupos de pestañas - descargar grupos completos
• Restauración de posición en YouTube - reanudar donde lo dejó
• Restauración de posición de desplazamiento al recargar pestañas
• Notificaciones de descarga automática
• Cerrar pestañas duplicadas - deduplicación con un clic en la ventana emergente
• Búsqueda de pestañas - filtro en vivo por título o URL
• Información sobre uso de memoria - pase el cursor para ver el desglose de memoria por pestaña
• Aviso de suspensión - advertencia suave de 3 segundos antes de la descarga automática
• Modo de panel lateral - abra la interfaz en la barra lateral persistente del navegador en lugar de la ventana emergente (permanece visible al cambiar de pestañas)
• Importar/exportar configuración - haga copia de seguridad y restaure listas blancas, listas negras y sesiones en JSON al portapapeles
• Asistente de configuración interactivo - incorporación guiada de 6 pasos al instalar (vuelva a ejecutarlo en cualquier momento desde Opciones)
• Informe de errores anónimo opcional (opt-in) - nos ayuda a corregir errores mientras protege su privacidad
• Soporte multilingüe
• Apertura automática del registro de cambios - vea las novedades en las actualizaciones (solo versiones menores/mayores)

💡 CÓMO FUNCIONA:
TabRest usa la API nativa tabs.discard() de Chrome. Las pestañas descartadas:
• Permanecen visibles en la barra de pestañas
• Conservan la posición de desplazamiento y los datos de formulario
• Se recargan al instante al hacer clic
• Liberan memoria mientras están inactivas

🔒 PRIVACIDAD:
• Sin recopilación de datos de forma predeterminada
• Informe de errores anónimo opcional (desactivado de forma predeterminada, totalmente transparente)
• Cuando está habilitado: errores anonimizados, PII (URLs, emails, IPs) redactado, máximo 100 reportes automáticos/día, deduplicación + muestreo para reducir ruido
• Toda la configuración almacenada localmente en su dispositivo
• Código abierto: https://github.com/lamngockhuong/tabrest

🌐 SITIO WEB: https://tabrest.ohnice.app

Hecho con ❤️ para los coleccionistas de pestañas de todo el mundo.
```

## Data Usage / Uso de datos

Información declarada en la pestaña "Prácticas de privacidad" de Chrome Web Store.

### Data Types Collected / Tipos de datos recopilados

Marque **ninguna** de las siguientes categorías. TabRest no recopila ninguna de ellas.

| Categoría                                    | ¿Recopilado? | Motivo                                                                    |
| -------------------------------------------- | ------------ | ------------------------------------------------------------------------- |
| Información de identificación personal (PII) | No           | No se recopilan nombres, correos electrónicos ni direcciones              |
| Información de salud                         | No           | No aplicable                                                              |
| Información financiera/de pago               | No           | Extensión gratuita, sin transacciones                                     |
| Información de autenticación                 | No           | No se recopilan contraseñas ni PIN                                        |
| Comunicaciones personales                    | No           | No se leen correos electrónicos ni mensajes                               |
| Ubicación                                    | No           | No se usa IP ni GPS                                                       |
| Historial web                                | No           | Las URL/títulos se procesan solo localmente, nunca se transmiten          |
| Actividad del usuario                        | No           | Sin registro de clics/teclas; marcas de tiempo LRU almacenadas localmente |
| Contenido del sitio web                      | No           | El contenido de la página no se lee ni se transmite                       |

> **Nota importante:** TabRest accede a las URL/títulos de las pestañas (mediante el permiso `tabs`) y lee el estado de los formularios (mediante `scripting`), pero todo se procesa localmente en el dispositivo y **nunca abandona la máquina del usuario**. Cuando un usuario opta por el informe de errores anónimo, solo se envían a Sentry trazas de pila saneadas y metadatos - sin URL, PII ni contenido de página.

### Disclosures / Declaraciones

Marque las 3:

- No vendo ni transfiero los datos del usuario a terceros (fuera de los casos de uso aprobados - Sentry actúa como procesador, no como venta de datos)
- No uso ni transfiero los datos del usuario para fines ajenos al propósito único
- No uso ni transfiero los datos del usuario para determinar la solvencia crediticia ni para fines de préstamo
