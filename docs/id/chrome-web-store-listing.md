## Full Description / Deskripsi Lengkap

Teks pemasaran panjang yang digunakan di kolom deskripsi Chrome Web Store.

```text
TabRest secara otomatis membongkar tab browser yang tidak aktif untuk membebaskan memori dan menjaga komputer Anda tetap berjalan dengan lancar.

🎯 SEMPURNA UNTUK:
• Pengguna dengan banyak tab terbuka
• Komputer dengan RAM terbatas
• Siapa saja yang menginginkan pengalaman browsing lebih cepat

✨ FITUR UTAMA:

📌 Pembongkaran Otomatis Cerdas
• Timer tidak aktif yang dapat dikonfigurasi (5 menit hingga 4 jam)
• Pemicu ambang memori (penggunaan RAM 60-95%)
• Batas memori per tab (membongkar tab yang menggunakan JS heap >100MB-1GB)
• Opsi bongkar saat startup untuk membebaskan memori saat browser dibuka
• Lewati saat offline - tidak membongkar tab saat jaringan tidak tersedia

⌨️ Kontrol Manual & Pintasan Keyboard
• Bongkar tab saat ini: Alt+Shift+D
• Bongkar tab lain: Alt+Shift+O
• Bongkar tab di sebelah kanan: Alt+Shift+Right
• Bongkar tab di sebelah kiri: Alt+Shift+Left
• Menu konteks klik kanan untuk akses cepat

🛡️ Fitur Perlindungan
• Daftar putih domain agar tidak pernah dibongkar
• Tunda tab individual (30 menit - 2 jam)
• Tunda seluruh domain untuk sementara
• Lewati tab dengan audio/video yang sedang diputar
• Lewati tab dengan formulir yang belum disimpan
• Opsi proteksi tab tersemat

📊 Statistik & Pemantauan
• Tampilan penggunaan RAM langsung di popup
• Lacak total tab yang dibongkar
• Lacak total memori yang dihemat
• Indikator visual (💤) pada judul tab yang dibuang

🔧 Fitur Tambahan
• Dukungan grup tab - bongkar seluruh grup
• Pemulihan posisi YouTube - lanjutkan dari tempat Anda berhenti
• Pemulihan posisi gulir saat tab dimuat ulang
• Notifikasi bongkar otomatis
• Tutup tab duplikat - deduplikasi satu klik di popup
• Pencarian tab - filter langsung berdasarkan judul atau URL
• Tooltip estimasi memori - arahkan kursor untuk melihat rincian memori per tab
• Toast peringatan penangguhan - peringatan lembut 3 detik sebelum bongkar otomatis
• Mode panel samping - buka UI di bilah samping browser yang persisten, bukan popup (tetap terlihat saat Anda berpindah tab)
• Impor/ekspor pengaturan - cadangkan dan pulihkan daftar putih, daftar hitam, dan sesi ke JSON clipboard
• Wizard penyiapan interaktif - orientasi 6 langkah terpandu saat instalasi (jalankan kembali kapan saja dari Opsi)
• Laporan kesalahan anonim opsional (opt-in) - membantu kami memperbaiki bug sambil melindungi privasi Anda
• Dukungan multibahasa
• Buka log perubahan otomatis - lihat hal baru pada pembaruan (khusus rilis minor/mayor)

💡 CARA KERJANYA:
TabRest menggunakan API chrome.tabs.discard() bawaan Chrome. Tab yang dibuang:
• Tetap terlihat di bilah tab Anda
• Mempertahankan posisi gulir dan data formulir
• Dimuat ulang seketika saat diklik
• Membebaskan memori saat tidak aktif

🔒 PRIVASI:
• Tidak ada pengumpulan data secara default
• Laporan kesalahan anonim opsional (dinonaktifkan secara default, sepenuhnya transparan)
• Saat diaktifkan: kesalahan dianonimkan, PII (URL, email, IP) dihapus, maksimal 100 laporan otomatis/hari, deduplikasi + sampling untuk mengurangi noise
• Semua pengaturan disimpan secara lokal di perangkat Anda
• Sumber terbuka: https://github.com/lamngockhuong/tabrest

🌐 SITUS WEB: https://tabrest.ohnice.app

Dibuat dengan ❤️ untuk para penimbun tab di mana saja.
```

## Data Usage / Penggunaan Data

Informasi yang diungkapkan di tab "Praktik privasi" Chrome Web Store.

### Data Types Collected / Jenis data yang dikumpulkan

Centang **tidak ada** dari kategori berikut. TabRest tidak mengumpulkan satu pun dari kategori tersebut.

| Kategori                                            | Dikumpulkan? | Alasan                                                                              |
| --------------------------------------------------- | ------------ | ----------------------------------------------------------------------------------- |
| Informasi yang dapat mengidentifikasi pribadi (PII) | Tidak        | Tidak ada nama, email, atau alamat yang dikumpulkan                                 |
| Informasi kesehatan                                 | Tidak        | Tidak berlaku                                                                       |
| Informasi keuangan / pembayaran                     | Tidak        | Ekstensi gratis, tidak ada transaksi                                                |
| Informasi autentikasi                               | Tidak        | Tidak ada kata sandi / PIN yang dikumpulkan                                         |
| Komunikasi pribadi                                  | Tidak        | Tidak ada pembacaan email atau pesan                                                |
| Lokasi                                              | Tidak        | Tidak menggunakan IP atau GPS                                                       |
| Riwayat web                                         | Tidak        | URL/judul diproses secara lokal saja, tidak pernah dikirimkan                       |
| Aktivitas pengguna                                  | Tidak        | Tidak ada pencatatan klik/penekanan tombol; stempel waktu LRU disimpan secara lokal |
| Konten situs web                                    | Tidak        | Konten halaman tidak dibaca atau dikirimkan                                         |

> **Catatan penting:** TabRest mengakses URL/judul tab (melalui izin `tabs`) dan membaca status formulir (melalui `scripting`), tetapi semuanya diproses secara lokal di perangkat dan **tidak pernah meninggalkan mesin pengguna**. Saat pengguna memilih untuk mengaktifkan laporan kesalahan anonim, hanya stack trace yang telah disanitasi dan metadata yang dikirim ke Sentry - tanpa URL, PII, atau konten halaman.
