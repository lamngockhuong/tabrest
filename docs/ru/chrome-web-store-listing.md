## Full Description / Полное описание

Расширенный маркетинговый текст для поля описания в Chrome Web Store.

```text
TabRest автоматически выгружает неактивные вкладки браузера, чтобы освободить память и поддерживать стабильную работу компьютера.

🎯 ИДЕАЛЬНО ПОДХОДИТ ДЛЯ:
• Пользователей с большим количеством открытых вкладок
• Компьютеров с ограниченным объёмом RAM
• Всех, кто хочет ускорить работу браузера

✨ КЛЮЧЕВЫЕ ВОЗМОЖНОСТИ:

📌 Умная автоматическая выгрузка
• Настраиваемый таймер бездействия (от 5 минут до 4 часов)
• Срабатывание по порогу памяти (60-95% использования RAM)
• Ограничение памяти на вкладку (выгрузка вкладок с JS heap >100 МБ - 1 ГБ)
• Выгрузка при запуске - освобождение памяти при открытии браузера
• Пропуск при отсутствии сети - не выгружать вкладки при недоступности интернета

⌨️ Ручное управление и сочетания клавиш
• Выгрузить текущую вкладку: Alt+Shift+D
• Выгрузить другие вкладки: Alt+Shift+O
• Выгрузить вкладки справа: Alt+Shift+Right
• Выгрузить вкладки слева: Alt+Shift+Left
• Контекстное меню для быстрого доступа

🛡️ Функции защиты
• Белый список доменов - никогда не выгружать
• Откладывать отдельные вкладки (30 мин - 2 часа)
• Временно откладывать целые домены
• Пропускать вкладки с воспроизведением аудио/видео
• Пропускать вкладки с несохранёнными формами
• Защита закреплённых вкладок

📊 Статистика и мониторинг
• Отображение текущего использования RAM во всплывающем окне
• Отслеживание общего числа выгруженных вкладок
• Отслеживание общего объёма сэкономленной памяти
• Визуальный индикатор (💤) в заголовках сброшенных вкладок

🔧 Дополнительные возможности
• Поддержка групп вкладок - выгрузка целых групп
• Восстановление позиции YouTube - продолжить с места остановки
• Восстановление позиции прокрутки при перезагрузке вкладок
• Уведомления об автоматической выгрузке
• Закрытие дублирующихся вкладок - дедупликация одним щелчком во всплывающем окне
• Поиск по вкладкам - фильтрация в реальном времени по заголовку или URL
• Подсказка с оценкой памяти - наведите курсор, чтобы увидеть разбивку памяти по вкладке
• Предупреждение о выгрузке - мягкое уведомление за 3 секунды до автоматической выгрузки
• Режим боковой панели - открыть интерфейс в постоянной боковой панели браузера вместо всплывающего окна (остаётся видимым при переключении вкладок)
• Импорт/экспорт настроек - резервное копирование и восстановление белых списков, чёрных списков и сессий в JSON через буфер обмена

• Поддержка нескольких языков
• Автоматическое открытие журнала изменений - просмотр новшеств при обновлении (только для минорных/мажорных выпусков)

💡 КАК ЭТО РАБОТАЕТ:
TabRest использует нативный API Chrome chrome.tabs.discard(). Сброшенные вкладки:
• Остаются видимыми в строке вкладок
• Сохраняют позицию прокрутки и данные формы
• Мгновенно перезагружаются при нажатии
• Освобождают память в неактивном состоянии

🔒 КОНФИДЕНЦИАЛЬНОСТЬ:
• По умолчанию данные не собираются
• Необязательная анонимная отправка отчётов об ошибках (отключена по умолчанию, полностью прозрачная)
• При включении: ошибки анонимизируются, PII (URL, email, IP) удаляются, максимум 100 автоматических отчётов/день, дедупликация + сэмплирование для снижения шума
• Все настройки хранятся локально на Вашем устройстве
```

## Data Usage / Использование данных

Сведения, раскрываемые во вкладке "Политика конфиденциальности" в Chrome Web Store.

### Data Types Collected / Типы собираемых данных

Не отмечайте **ни одну** из следующих категорий. TabRest не собирает ни одну из них.

| Категория                               | Собирается? | Причина                                                                      |
| --------------------------------------- | ----------- | ---------------------------------------------------------------------------- |
| Персональные данные (PII)               | Нет         | Имя, электронная почта и адрес не собираются                                 |
| Информация о здоровье                   | Нет         | Не применимо                                                                 |
| Финансовая информация / данные платежей | Нет         | Бесплатное расширение, транзакции отсутствуют                                |
| Данные аутентификации                   | Нет         | Пароль / PIN не собираются                                                   |
| Личные сообщения                        | Нет         | Чтение электронной почты или сообщений не производится                       |
| Местоположение                          | Нет         | IP и GPS не используются                                                     |
| История посещений                       | Нет         | URL/заголовки обрабатываются только локально и никогда не передаются         |
| Активность пользователя                 | Нет         | Нажатия клавиш/кликов не регистрируются; LRU-метки времени хранятся локально |
| Содержимое веб-страниц                  | Нет         | Содержимое страниц не читается и не передаётся                               |

> **Важное примечание:** TabRest получает доступ к URL и заголовкам вкладок (через разрешение `tabs`) и считывает состояние форм (через `scripting`), однако всё это обрабатывается локально на устройстве и **никогда не покидает компьютер пользователя**. Когда пользователь включает анонимную отправку отчётов об ошибках, в Sentry отправляются только очищенные трассировки стека и метаданные - без URL, персональных данных и содержимого страниц.
